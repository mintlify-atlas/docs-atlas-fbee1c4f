Searching algorithms are methods used to find a specific item in a set of data.

## Searching Algorithms

| Name                     | Time        | Space  | Overview                                    |
| ------------------------ | ----------- | ------ | ------------------------------------------- |
| [Linear Search](./linear-search) | `O(n)`      | `O(1)` | Checks each element one by one.             |
| [Binary Search](./binary-search) | `O(log(n))` | `O(1)` | Searches sorted data by halving range.      |
| [Depth-First Search (DFS)](./dfs) | `O(n)`      | `O(h)` | Explores tree/graph one path before others. |
| [Breadth-First Search (BFS)](./bfs) | `O(n)`      | `O(n)` | Explores tree/graph level by level.         |
